# Деплой на сервери expo

## Завдання

1. Переконатися, що ваш проект працює локально
2. Налаштувати конфігурацію проекту для публікації.
3. Забезпечити коректність налаштувань для публікації.
4. Опублікувати проект на сервери Expo.
5. Перевірити доступність вашого додатку.
6. Виправити можливі помилки та повторити публікацію при необхідності.
